# Todolist app

## Technologies
- AWS
- Tailwind
- NextJS

## Features
Basic todo app with
- Authentication
- Authorization
- Creating todos
- Editing todos
- Deleting todos
- Cropping and saving user avatar

## AWS Services
- Cognito for authentication
- API Gateway & Lambda functions for app logic
- DynamoDB for data persistence
- S3 for avatar uploads
